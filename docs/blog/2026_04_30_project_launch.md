# SiliconTrace Open 项目启动：开源 EDA 赋能中国芯片教育

> 作者: SiliconTrace Open 团队
> 日期: 2026-04-30
> 标签: #开源EDA #RISC-V #SKY130 #芯片教育

---

今天，我们正式发布 **SiliconTrace Open** 项目——一个完全开源的数字集成电路设计流程，从 RTL 代码到可制造的 GDSII 版图，再到测试 PCB 板，全流程基于开源工具链实现。

## 项目动机

### 芯片教育的困境

作为一名芯片设计专业的学生，我在学习过程中深刻体会到当前芯片教育面临的困境：

1. **工具壁垒**: 商业 EDA 工具（Synopsys、Cadence、Mentor Graphics）价格昂贵，学校难以购买完整授权
2. **实践缺失**: 学生只能在仿真环境中学习，无法接触到真实的制造流程
3. **知识碎片化**: 综合、布局、布线、时序分析等知识点分散在不同课程中，缺乏系统性实践
4. **国产替代**: 国产 EDA 工具（如 iEDA）已经具备相当实力，但缺乏完整的教学案例

### 开源 EDA 的机遇

近年来，开源 EDA 生态发生了翻天覆地的变化：

- **Yosys**: 开源逻辑综合工具，支持 Verilog 综合
- **OpenROAD**: 开源数字后端全流程工具
- **iEDA**: 中科院计算所开发的国产开源数字后端平台
- **SKY130 PDK**: SkyWater 130nm 开源工艺库
- **Google + SkyWater**: 首个开源芯片制造计划

这些工具的成熟，使得"从 RTL 到芯片"的完整流程可以在开源环境下实现。

## 项目目标

SiliconTrace Open 的目标是：

1. **教育**: 为高校学生提供完整的芯片设计实践平台
2. **开源**: 所有设计文件、脚本、文档完全开源
3. **国产**: 优先使用国产开源工具（iEDA）
4. **完整**: 从 RTL 到 GDSII，再到测试 PCB，全流程覆盖

## 工具链介绍

### Yosys - 开源逻辑综合

Yosys 是 Claire Wolf 开发的开源综合框架，支持：

- Verilog 2005 语法
- 多种优化策略
- 面向不同工艺的映射
- 形式验证集成

```bash
# 使用 Yosys 综合 PicoRV32
yosys -p "read_verilog picorv32.v; synth -top picorv32; abc -liberty sky130_fd_sc_hd.lib"
```

### iEDA - 国产开源数字后端

iEDA 是中科院计算所开发的数字后端 EDA 平台，包括：

- **iPL**: 布局引擎
- **iCTS**: 时钟树综合
- **iRT**: 布线引擎
- **iSTA**: 静态时序分析
- **iDRC**: 设计规则检查

iEDA 的优势：

1. 完全开源，可学习后端算法实现
2. 国产自主可控
3. 支持 SKY130 等开源 PDK
4. 社区活跃，持续更新

### SKY130 PDK - 开源工艺库

SkyWater 130nm 工艺是首个完全开源的半导体工艺：

- 特征尺寸: 130nm
- 金属层: 5 层
- 电源电压: 1.8V / 3.3V
- 标准单元库: sky130_fd_sc_hd (高密度)

### KiCad - 开源 PCB 设计

KiCad 是功能强大的开源 PCB 设计工具：

- 原理图设计
- PCB 布局布线
- 3D 可视化
- 制造文件输出

## 第一个里程碑

### RTL 到 GDSII 流程验证

我们成功在开源工具链上完成了 PicoRV32 的 RTL 到 GDSII 流程：

```
PicoRV32 RTL (Verilog)
    │
    ▼
Yosys 综合
    │  标准单元: sky130_fd_sc_hd
    │  面积: ~12800 um^2
    │  频率: ~85 MHz
    ▼
iEDA 后端
    │  Floorplan: 200x200 um
    │  Placement: 密度 70%
    │  CTS: Skew < 500ps
    │  Routing: DRC Clean
    ▼
GDSII 输出
    │  文件: picorv32.gds
    │  大小: ~2MB
    ▼
DRC 检查 (KLayout)
    │  违规数: 0
    ▼
制造就绪
```

### 关键数据

| 指标 | 值 |
|------|-----|
| 标准单元数 | ~4800 |
| 触发器数 | ~287 |
| 核心面积 | 0.012 mm^2 |
| 目标频率 | 50 MHz |
| 实测频率 | 85 MHz |
| 功耗估算 | ~5 mW |

### 遇到的挑战

在项目实施过程中，我们遇到了不少挑战：

1. **工具兼容性**: Yosys 和 iEDA 之间的接口格式需要适配
2. **PDK 配置**: SKY130 PDK 的版本管理和配置比较复杂
3. **时序约束**: 从 SDC 到 iEDA 的约束转换
4. **学习曲线**: 开源工具的文档相对较少
