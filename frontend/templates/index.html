MCP 交互上传接口

POST /api/upload/mcp
Content-Type: application/json

{
  "action": "import",
  "type": "rtl",          // rtl | footprint | symbol
  "filename": "my_design.v",
  "content": "...",       // 文件内容
  "encoding": "text",     // text | base64
  "design_name": "my_design"  // 仅 RTL 需要
}

示例 (curl):
curl -X POST http://localhost:5000/api/upload/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "action": "import",
    "type": "rtl",
    "filename": "counter.v",
    "content": "module counter(...); endmodule",
    "encoding": "text",
    "design_name": "counter"
  }'
