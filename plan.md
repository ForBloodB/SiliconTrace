【项目名称】硅迹开源（SiliconTrace Open）

【项目定位】
由在校大三学生独立发起并运营的「全开源数字IC设计工作室（One-Person Studio）」。项目以Yosys（开源综合工具）、iEDA（国产开源数字后端平台）、KiCad（开源PCB设计）、SKY130 PDK为核心支柱，构建覆盖「数字芯片设计 → 版图Sign-off → 封装/PCB协同 → 板级测试验证」的完整数字IC设计与验证流程。项目不以商业利润为首要目标，而是以「验证一条中国人可以零成本走通的芯片设计之路」为使命，产出真实可运行的芯片+硬件作品，并将全流程工程经验以开源形式反哺社区。

【市场痛点】

1. 国产EDA「看得懂用不起」，开源工具「装得上走不通」。华大九天等国产商业EDA对高校学生存在授权门槛；而Yosys、iEDA、KiCad等开源工具虽然零成本，但缺乏面向中文用户的系统化整合指南，学生在安装、配置PDK、打通数据接口环节频频受阻，最终放弃。
2. 高校教育「严重割裂」：芯片设计与板级验证脱节。集成电路课程止步于GDSII交付，电子系统设计课程止步于PCB焊接。学生同时懂芯片后端实现和PCB信号完整性的人凤毛麟角，而这正是流片后调试成功率低的核心原因。
3. 中小型硬件团队缺少「全栈验证」外援。高校竞赛团队、智能硬件初创公司、个人开发者，需要的不只是「生成一个GDSII」，而是「这颗芯片在我自己的板子上能不能跑起来」的确定性答案。传统外包模式下，IC设计和PCB设计由不同团队完成，责任边界模糊，调试期长达数月。
4. 开源芯片运动缺少「最后一公里」服务。全球开源芯片社区（Open MPW、Efabless）聚焦于「如何把GDSII送出去流片」，但流片回来后的芯片Bring-up、测试程序开发、测试载板设计环节几乎空白。大量开源芯片回片后因没有配套测试环境而被闲置。

【解决方案】
硅迹开源提出的解法：「一人一栈，工具全开源，交付可验证」。
核心理念：不以单一环节（IC/PCB）作为交付物，而是以「一颗在我自己设计的板子上跑通了的芯片」作为唯一交付标准。
具体实现路径：
• 数字线：Verilog RTL → Yosys综合 → iEDA布局布线/STA → SKY130 Sign-off → GDSII流片
• 系统线：芯片设计阶段即同步启动KiCad测试载板规划（引脚分配、电源架构、高速拓扑）
• 验证线：回片后使用自制测试板 + 自研测试程序完成Bring-up、功能/性能验证，输出测试报告
自动化层：使用Python/Tcl脚本打通Yosys↔iEDA之间的网表、约束、坐标数据交换，形成可复用的开源脚本库。

【商业模式】
阶段一：开源内容 + 社区影响力（当前）
• 在B站/知乎/CSDN/GitHub上发布「全开源数字IC设计实战」系列内容：从Yosys综合、iEDA后端到KiCad测试板的完整踩坑记录。
• 将自动化脚本、KiCad封装库、测试板模板以MIT协议开源至GitHub。
• 目标：积累技术社区的Star、关注度和个人技术品牌。
阶段二：轻量级技术服务（能力成熟期）
• 面向校内及周边高校的小型硬件团队，提供：
  - KiCad高速PCB代画/审图（四层以内，含SI/PI基础优化）
  - Yosys+iEDA数字后端实现服务（综合、STA分析、GDSII生成）
  - 芯片测试载板+Bring-up支持（新回片芯片的上电、基础测试程序）
• 定价策略：低于市场价30%-50%，但要求客户允许匿名化后作为案例展示。
阶段三：标准化IP + 评估套件授权
• 将高频需求模块（如SKY130通用测试载板、RISC-V最小系统PCB模板）打包为可授权的开源硬件IP，通过Tindie/淘宝开源硬件店销售PCB空板或套件。
阶段四：工具链培训与技术咨询
• 面向有国产替代需求的中小企业、高校实验室，提供「开源EDA工作流搭建」的远程培训与环境配置服务。

【核心服务与功能】
一、数字ASIC设计服务
(1) 数字后端全流程实现：基于Yosys+iEDA工作流，完成从Verilog RTL综合、布局布线、STA时序分析到SKY130 PDK Sign-off的完整数字后端流程，产出可交付的GDSII文件。
(2) 开源IP后端优化：针对RISC-V核（如PicoRV32、SERV）等开源IP，进行面向SKY130工艺的物理实现与时序收敛。
二、PCB与系统级服务
(3) 芯片测试载板设计：使用KiCad为回片芯片设计专用测试板，支持QFN/BGA封装，包含电源、调试接口、外设连接。
(4) 高速PCB Layout：四层以内多层板设计，含信号完整性（SI）与电源完整性（PI）基础优化。
三、验证与Bring-up服务
(5) 回片测试支持：提供芯片上电调试、基础功能/性能测试（ADC/DAC/电源指标）、边界扫描与失效定位。
(6) 自动化脚本库：打通Yosys↔iEDA之间的数据接口，实现「一键式」网表/约束转换。

【核心优势】

1. 全开源零成本壁垒：整个工具链（Yosys/iEDA/KiCad/SKY130 PDK）完全免费，启动成本趋近于零（一台Linux工作站+PCB打样费即可开始），无授权风险，作品可完全公开分享。
2. 国内首批「iEDA + KiCad」全栈实践者：iEDA是国产开源数字后端领域的新星，KiCad是全球最活跃的开源PCB工具，但将两者与Yosys、SKY130打通整合形成完整工作流的个人/团队在国内极为稀缺。先发者的踩坑经验本身就是高价值资产。
3. 交付物天然具备「可验证性」：传统IC设计外包交付的是GDS文件，客户不知道能不能用；硅迹开源的交付标准是「在我自己设计的测试板上跑通的芯片系统」，这是最高级别的质量背书。
4. 契合开源芯片运动的时代红利：Google Open MPW、Efabless、国内「芯原杯」等渠道正在降低流片门槛。硅迹开源踩通的这条路，不仅是个人能力证明，也可以成为后来者跟随的「中文区开源芯片设计路线图」。

【项目规划】
• 第一阶段：toolchain闭环验证

- 完成Yosys+iEDA+SKY130数字后端工作流跑通：以开源RISC-V核（PicoRV32）为目标，完成综合、布局布线、STA分析，产出GDSII。
- 使用KiCad设计上述芯片的测试载板（原理图+四层PCB Layout）。
- 建立GitHub工程仓库，开始撰写系列技术博客。
- 里程碑：拥有1个完整的开源数字IC设计工程，具备提交MPW流片的技术准备。
• 第二阶段：流片实战 + 社区建设
- 通过Efabless Open MPW或国内开源Shuttle提交至少1次真实流片。
- 同步完成测试载板的打样、焊接与基础程序编写。
- 芯片回片后完成Bring-up测试，录制完整调试过程作为核心作品集素材。
- 在B站/知乎发布「如何用全开源工具设计一颗数字芯片」系列视频。
- 里程碑：手上有1颗回片芯片 + 1套跑通的测试系统 + 初步的技术社区影响力。
• 第三阶段：品牌输出 + 成果转化
- 整理全部脚本与模板，发布「硅迹开源工具箱」（开源脚本库+KiCad封装库+测试板模板）。
- 承接1-2个轻量级数字后端设计订单，验证商业可行性。

【负责人背景】
• 学历背景：某高校电子科学与技术专业大三在读本科生。
• 理论基础：系统修完《半导体物理》《数字集成电路设计》《信号与系统》《微电子工艺》等核心课程，对CMOS器件物理、数字电路设计有深入理解。
• 数字设计实践：熟悉Verilog HDL数字设计方法，掌握Yosys开源综合工具与iEDA数字后端平台，能独立完成从RTL综合到布局布线、STA时序分析的完整流程。
• 硬件系统实践：长期使用KiCad完成多层PCB设计（最高四层），具备从原理图设计、Layout、DRC到贴片调试的完整硬件工程能力；对高速信号完整性、电源完整性有基础认知。
• 工具链整合能力：熟悉Python/Tcl脚本编程，致力于打通Yosys↔iEDA、iEDA↔KiCad之间的数据接口，追求「一键式」工作流自动化。
• 开源社区参与：活跃于开源芯片/开源硬件社区，认同开源文化，致力于将芯片设计门槛从「大企业专属」拉低到「个人开发者可触及」。
• 项目愿景：证明一个普通的中国大学生，不依赖任何商业软件License，仅凭开源工具和一台电脑，也能走完从芯片设计到板级验证的完整链路。硅迹开源不仅是个人技术修行，也是为后来者铺设的一块垫脚石。
