# PicoRV32 形式验证
# 硅迹开源 (SiliconTrace Open)

## 概述

本目录包含 PicoRV32 RISC-V 核心的形式验证文件。

形式验证使用数学方法证明设计的正确性，而非通过仿真激励来测试。
主要验证内容包括：
- 寄存器传输级 (RTL) 属性检查
- 不变量验证
- 安全性属性验证

## 工具

- **SymbiYosys**: 开源形式验证前端
- **Yosys**: RTL 综合引擎（形式验证后端依赖）

## 文件说明

| 文件 | 说明 |
|------|------|
| `picorv32.sby` | SymbiYosys 配置文件 |
| `properties.v` | 形式验证属性定义 |
| `run_formal.sh` | 形式验证运行脚本 |

## 使用方法

```bash
# 运行形式验证
cd tests/formal
bash run_formal.sh
```

## 验证属性示例

1. **复位后状态**: 复位释放后，程序计数器应为 PROGADDR_RESET
2. **指令有效性**: 每个解码的指令应为有效的 RV32I 指令
3. **存储器访问对齐**: 非对齐存储器访问应触发异常
4. **寄存器文件一致性**: 寄存器 x0 始终为 0
